Airline Reservation System

A robust and efficient management system designed to streamline the process of flight bookings, passenger record management, and ticketing operations. This project showcases strong foundational programming logic and core computer science concepts.

Key Features
- **Flight Management:** View available flights, schedules, and seat availability.
- **Ticket Booking:** Seamlessly book tickets for passengers by collecting required details.
- **Ticket Cancellation:** Allows users to cancel existing reservations and updates seat availability dynamically.
- **Passenger Records:** securely stores and manages passenger information.
- **Automated Fare Calculation:** Calculates ticket prices based on flight class and destination.

🛠️ Tech Stack
- **Language:** C++.
- **Core Concepts:** Object-Oriented Programming (OOP), File Handling, Data Structures, Control Flows.

💡 What I Learned
Building this system significantly improved my problem-solving skills. It gave me hands-on experience with managing data states, implementing complex logic loops, and utilizing file handling to ensure data is saved persistently.
